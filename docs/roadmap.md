# TenderPilot AI - Product Roadmap

TenderPilot AI's development path focuses on expanding geographic portal reach, integrating deep Gremlin Graph heuristics, and introducing self-learning optimization loops.

---

## Phase 1: Foundation (Q3 2026)
* **Core Agent Pipeline**: Fully validate the 8-agent Semantic Kernel pipeline.
* **Integrations**: Expand Microsoft Graph endpoints to support Teams alerts, PowerBI analytics dashboards, and Jira tickets.
* **Data Sources**: Integrate specialized PDF parsers to extract bid tables and pricing sheets.

## Phase 2: Knowledge Graph Expansion (Q1 2027)
* **Cross-Border Mapping**: Link tender databases across geographies (e.g. tracking how an EU public bid compares with municipal Smart City bids in India).
* **Price Suggestion Engine**: Implement regression models matching historical bid win amounts with active inflation rates to predict winning price bands.
* **Cosmos Gremlin Scaling**: Migrate graph structures to Azure Cosmos DB for Apache Gremlin to support millions of legal entity relationships.

## Phase 3: Self-Learning & Optimization (Q3 2027)
* **Automated Prompt Optimization**: Implement LLM-in-the-loop prompt engineering, where the Reflection Agent feeds proposal review comments back to Azure AI Foundry to auto-optimize system prompts.
* **Autonomous Login Engines**: Build robust session recovery loops that handle multi-factor SMS login inputs using secure OTP endpoints.
